在 `Python` 或者  `Go` 里，如果想交换两个变量的值可以直接一行搞定，非常方便

```Go
i, j = j, i
```

但是 `C` 里面没有这种写法，如果想交换两个变量一般都会声明个中间的临时变量，然后进行交换

```C
temp = i;
i = j;
j = temp;
```

但是有种比较秀操作的写法，根本不需要第三个变量的参与，只需要这两个变量本身即可

```C
i = i + j;
j = i - j;
i = i - j;
```
